    Hover

    @mixin Hover($HTDuration: .5s, $HOpacity: .7, $HShadow: null, $HDecoration: none, $HColor: Color-3) { ... }

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $HTDuration

    Éste valor determina la velocidad con la que se va a mostrar el efecto del Hover, cuando se tenga que aplicar este valor a un botón, éste tendrá que ser de .3s

    		Transition-duration.5s
    $HOpacity

    Éste valor determina opacidad que se le aplica al elemento cuando el Hover se activa

    		Opacity.7
    $HShadow

    Éste valor determina si se le aplica el box-shadow configurado por defecto al elemento cuando se activa el Hover, para realizar esa acción se le tiene que asignar el valor shadow

    		Box-shadownull
    $HDecoration

    Éste valor determina si se le aplica algún valor a la propiedad de text-decoration cuando se activa el Hover

    		Text-decorationnone
    $HColor

    Éste valor aplicar el color de la interacción al momento de activar el Hover, ésta propiedad solo admite valores de nuestra paleta de colores

    		ColorColor-3

    Example

    @include Hover(.7s, .5, shadow, underline, Color-3);

.item {

    transition: opacity .7s;

    &:hover {

        opacity: .5;
        box-shadow: 0px 0px 20px #000000;
        text-decoration: underline;
        color: #E3E3E3;

    }
}
